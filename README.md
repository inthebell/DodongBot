## ✨ 주요 기능

### 🍳 요리 변동상점
- 변동상점 요리를 자동 분석
- 추천 요리 자동 표시

### 🧪 연금 변동상점
- 연금 변동상점 자동 분석
- 룬 시세 계산

### 📚 동글랜드 위키
- 위키 자동 검색
- 관련 페이지 안내
  
### 🤝 동협 계산기
- 물의 결정 개수만 입력하면 총 동협 포인트 자동 계산

### ⭐ 레벨 계산기
- 경험치를 입력하면 현재 필요 레벨 계산

### ✅ 도동마을 입주 인증
- 도동마을 전용 인증 시스템
